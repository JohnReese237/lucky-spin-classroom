# 2026-05-27

## 抽奖结果信息框 - 姓名滚动停留功能

修改了 `src/index.css` 中的 `result-text-slide` 动画，让姓名滚动到最左侧时停留 1.5 秒。

**具体修改：**
1. `.marquee-text.scrolling .marquee-track` 动画时长从 `8s` 改为 `9.2s`
2. `@keyframes result-text-slide` 中起始停留百分比从 `4%` 改为 `16.3%`（9.2s × 16.3% = 1.5s）

**效果：** 抽奖结果卡片中的学生姓名滚动到最左侧时会暂停 1.5 秒，滚屏速度保持不变。
